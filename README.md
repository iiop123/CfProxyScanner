# 主要用途
扫描指定IP段中是否存在反代/端口转发cloudflare的IP

------------


### 使用方法
- 修改 `"data/tesips.txt" `文件里IP为你需要扫描的IP

- 在根目录执行 `npm i`安装依赖环境

- 执行 `node check80.mjs`扫描指定IP段80端口 OR 执行 `node check443.mjs`扫描指定IP段443端口

- 扫描完成后符合条件的IP将输出到 `data/ips.txt`文件
